# 技術Q&A - 2026-02-19

### `backend/target/` ディレクトリはコミットが必要か？

不要。`target/` は Maven のビルド生成物（コンパイル結果・jarファイルなど）が出力されるディレクトリで、ソースコードから自動生成される。

**理由:**
- `mvn compile` / `mvn package` を実行するたびに自動生成されるため、Git で管理する必要がない
- コミットしてしまうとリポジトリが肥大化し、他の開発者の環境でビルドした結果と衝突する可能性がある

**対処:**
`.gitignore` に `target/` を追加することで自動的に除外される。

```
# Java / Maven
target/
```

一度も `git add` していない（untracked）状態であれば `.gitignore` に追加するだけで効く。
誤ってコミットしてしまった場合は `git rm -r --cached target/` でインデックスから削除する必要がある。
