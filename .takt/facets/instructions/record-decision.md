承認された実装プランの決定事項を、対象の GitHub Issue にコメントとして記録してください。

**手順:**
1. これまでのステップから、確定した決定事項（方針・スコープ・対象ファイル・除外項目）を整理する
2. 決定事項を一時ファイル（例: `/tmp/decision.md`）に Markdown で書き出す
3. `gh issue comment <番号> --body-file /tmp/decision.md` で Issue に投稿する
4. 投稿に成功したら、コメントの URL を報告する

**注意:**
- 確定した事実のみを書く。検討中・未確定の内容を確定として書かない
- 投稿コマンドの結果を確認する。失敗した場合は理由を報告し `Cannot proceed` を選択する

投稿が完了したら `Recorded to issue` を選択してください。
