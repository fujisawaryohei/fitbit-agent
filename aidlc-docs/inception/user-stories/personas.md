# Personas — Fitbit Weight Management AI Agent

## Persona 1: ヘルスコンシャスな自己管理ユーザー（主ペルソナ）

**名前**: Ryo（30代・会社員）
**役割**: アプリの唯一のエンドユーザー（個人利用）

### 特徴
- Fitbitデバイスを日常的に装着し、歩数・心拍数・睡眠データを記録している
- 目標体重への達成意欲があるが、自分でカロリー計算や運動計画を立てるのが面倒
- AIチャットの操作に慣れており、自然言語でやり取りすることへの抵抗がない
- ジムには通っておらず、自宅でできる運動を求めている
- Fitbitアプリで食事ログを手動で記録することがある（毎日ではない）

### 目標・ニーズ
- 現在の体重・活動量データを踏まえた、現実的な目標体重プランが欲しい
- 食事制限と運動のバランスを取ったアドバイスが欲しい
- 「今日は1500kcal食べた」とチャットで伝えれば、それも考慮してもらいたい
- 過去の会話や目標設定を覚えていてほしい（毎回説明しなくていい）
- 運動メニューは種目名・回数だけでなく、やり方や初心者向けバリエーションも知りたい

### フラストレーション
- 自分でスプレッドシートにデータを入力・管理するのは続かない
- 汎用的な「1日1800kcalで運動しましょう」では物足りない
- 毎回同じ目標や体重を伝え直すのが煩わしい
